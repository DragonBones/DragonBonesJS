# 5.0 相对 4.5 格式变化

## Root
* 增加 "compatibleVersion" 属性。
* 删除 "isGlobal" 属性。

## Display
* "type" 属性增加 "boundingBox" 类型。
    * 增加包围盒数据。
* 增加 "subType" 属性。
    * 区分包围盒类型。
* 增加 "share" 属性。
    * 增加共享网格功能。
* 增加 "width" "height" 属性。
    * 描述包围盒尺寸。
* 增加 "path" 属性。
    * 区分与 name 用于真正标识贴图、子骨架或其他扩展显示对象的索引。

## Animation
* 增加 "zOrder" 属性。
    * 增加动画中调整插槽深度排序功能。

## Frame
* 增加 "events" 列表。
    * 增加事件参数功能。
* 删除 "event" 属性。
