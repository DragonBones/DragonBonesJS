## How to run
* $ `npm install`
* $ `npm run start`

## Make sure project structure like this:
```
Your project
    |-- libs
        |-- dragonBones
            |-- dragonBones.js
            |-- ...
        |-- pixi
            |-- pixi.js
            |-- ...
    |-- node_modules
        |-- ...
    |-- out
        |-- ...
    |-- resource
        |-- ...
    |-- src
        |-- ...
    |-- ...
```
