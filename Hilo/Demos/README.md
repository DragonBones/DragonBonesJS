## How to run
* $ `npm install`
* $ `npm run start`

## Make sure project structure like this:
```
Your project
    |-- libs
        |-- dragonBones
            |-- dragonBones.js
            |-- ...
        |-- phaser
            |-- phaser.d.ts
            |-- pixi.d.ts
            |-- p2.d.ts
            |-- ...
    |-- node_modules
        |-- ...
    |-- out
        |-- ...
    |-- resource
        |-- ...
    |-- src
        |-- ...
    |-- ...
```
